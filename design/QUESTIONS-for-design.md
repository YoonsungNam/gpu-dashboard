## A. 시안 내부 모순 (어느 쪽이 맞는지 확인 필요)

### A-1. Unit 구성 테이블 3번째 지표 컬럼 헤더 — 'GPU Util'? 'GPU AH'?
- Figma: 펼침 상세의 Unit 구성 컬럼 헤더가 `GPU / GPU WH / GPU Util / Slot`로 되어 있음 (3번째가 'GPU Util')
- 그런데 바로 위 KPI 스트립은 `GPU Util / GPU Util WH / GPU Util AH / Slot Util` 순서라, 3번째 컬럼은 비근무(AH) 지표로 보임
- **현재 구현**: 오타로 판단하고 `GPU AH`로 표기
- **질문**: 3번째 컬럼 헤더 확정 표기는?

### A-2. 펼침 상세 패널 — Overview와 GPU 활용 현황의 스펙이 서로 다름
- Overview 펼침(7104:10456~): KPI 카드 68px, 값 600/16 우정렬, 색 #239B2F/#FF4337
- GPU 활용 현황 펼침(7104:9750~): KPI 카드 90px, 값 600/24 좌정렬, 색 #145C1C/#D2362C
- **현재 구현**: 두 변형을 모두 구현(화면별로 다르게 렌더)
- **질문**: 의도된 분기인가요, 아니면 한쪽 기준으로 통일하나요?

### A-3. 저활용(빨강) 배지 배경색 — #FFE3E1 vs #FFD9D7
- Overview 랭크 테이블의 큰 배지는 `#FFE3E1`, GPU 활용 현황 행 칩·범례는 `#FFD9D7`로 샘플링됨
- **현재 구현**: 화면별로 각각 적용
- **질문**: 통일 색상이 있나요?

### A-4. 활용도 점검 카드의 'N Projects' 카운터 색
- 노드 트리에는 과제 액센트색(#007492/#5A49A6) 흔적, 실제 렌더 추출본은 중립 #565E66
- **현재 구현**: 추출본 기준 중립 #565E66 (숫자 500/16 + ' Projects' 400/16)
- **질문**: 액센트인가요, 중립인가요?

## B. 시안에 정의되지 않은 동작/규칙

### B-1. '저활용 회수 예상량' 섹션 노출 규칙
- 시안에는 학습 펼침과 Overview 저활용 행 펼침에 보임
- **현재 구현**: `회수 추정 데이터 존재 AND (학습 탭 OR 회수 수량 > 0)`일 때 노출
- **질문**: 정확한 노출 조건은? (추론 과제도 회수 대상이면 보여주나요?)

### B-2. 토큰 활용 현황 — 그룹 52개의 페이지네이션
- KPI에는 '서비스 그룹 52'인데 시안 테이블에는 페이지네이션이 없음
- **현재 구현**: 전체 그룹을 한 번에 렌더 (스크롤)
- **질문**: 52개 전부 스크롤인가요, 페이지네이션/무한스크롤인가요?

### B-3. 정렬(sort) 동작
- GPU 활용 현황 테이블의 4개 지표 컬럼에만 정렬 삼각형 글리프가 있음 (정적)
- **현재 구현**: 글리프만 표시, 실제 정렬 미동작
- **질문**: 클릭 정렬이 동작해야 하나요? 기본 정렬 기준은?

### B-4. 사이드바 접기(collapse) 버튼
- 헤더 우측에 접기 버튼(◀)이 그려져 있으나 접힌 상태 시안이 없음
- **현재 구현**: 표시만 (동작 없음)
- **질문**: 접힘 상태 시안이 나올 예정인가요?

### B-5. '지표 정의' 링크 동작
- 탑바 우측 'ⓘ 지표 정의' — 클릭 시 동작이 시안에 없음
- **질문**: 팝오버/모달/외부 링크 중 무엇인가요? 내용 시안이 있나요?

### B-6. 서비스 전체 모달 — 닫기 푸터
- 컴포넌트 마스터에는 60px 푸터+'닫기' 버튼이 있으나 인스턴스에서는 숨김 처리됨
- **현재 구현**: 푸터 없음 (X / Esc / 오버레이 클릭으로 닫기)
- **질문**: 푸터 숨김이 맞나요?
